# BaseBookvsMovie

A Farcaster Mini App on Base where users vote for their favorite book or film adaptation and earn CSM tokens.

## Features
- 20 classic book vs film adaptations
- 100 CSM token reward per vote
- 1 vote per title per day
- Real-time vote statistics
- Base App & Farcaster Mini App support
